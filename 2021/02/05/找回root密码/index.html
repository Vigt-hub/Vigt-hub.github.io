找回root密码

2021年02月05日 留言

记录如何找回root密码

  1. 进入开机界面，在界面中按 e 进入编辑界面。
  2. 进入编辑界面，找到以“Linux16”开头内容所在行，在行的最后面输入 init=/bin/sh
  3. 按快捷键 Ctrl+x 进入单用户模式
  4. 输入 mount -o remount,rw / （各个单词间有空格）
  5. 输入 passwd 回车，输入新密码回车,再次输入密码，显示passwd…..的样式，说明密码修改成功
  6. 输入 touch /.autorelabel （注意：touch与 /后面有一个空格），完成后回车键
  7. 输入 exec /sbin/init 等待系统重启即可
