EL表达式

2021年04月07日 留言
  1. Expression LanguageEL表达式语言,替换和简化jsp页面中java代码的编写,语法${表达式}。
  2. JavaServer Pages Tag LibraryJSP标准标签库,是由Apache组织提供的开源的免费的jsp标签,格式为<标签>。
  3. 三层架构：软件设计架构

    1.EL的使用

    1.1 运算：

  • 算数运算符： + - * /(div) %(mod)

  • 比较运算符： > < >= <= == !=

  • 逻辑运算符： &&(and) ||(or) !(not)

  • 空运算符： empty,用于判断字符串、集合、数组对象是否为null或者长度是否为0。例如${empty list}:判断字符串、集合、数组对象是否为null或者长度为0。${not empty str}:表示判断字符串、集合、数组对象是否不为null并且长度>0。

1.2 获取值

  1. el表达式只能从域对象中获取值

  2. 语法：

    1. ${域名称.键名}：从指定域中获取指定键的值

      • 域名称：
        1. pageScope –> pageContext
        2. requestScope –> request
        3. sessionScope –> session
        4. applicationScope –> application（ServletContext）
      • 举例：在request域中存储了name=张三
      • 获取：${requestScope.name}

    2. ${键名}：表示依次从最小的域中查找是否有该键对应的值，直到找到为止。

    3. 获取对象、List集合、Map集合的值

      1. 对象：${域名称.键名.属性名},本质上会去调用对象getter方法

      2. List集合：${域名称.键名[索引]}

      3. Map集合：${域名称.键名.key名称},${域名称.键名["key名称"]}

2.JSTL

用于简化和替换jsp页面上的java代码

2.1 使用步骤

  1. 导入jstl相关jar包

  2. 引入标签库：taglib指令： <%@ taglib %>

  3. 使用标签

    2.2 常用的JSTL标签

  4. if:相当于java代码的if语句

    1. 属性：
      • test是必须属性，接收boolean表达式。如果表达式为true，则显示if标签体内容，如果为false，则不显示标签体内容
      • 一般情况下，test属性值会结合el表达式一起使用
    2. 注意：
      • c:if标签没有else情况，想要else情况，则可以在定义一个c:if标签

  5. choose:相当于java代码的switch语句

    1. 使用choose标签声明，相当于switch声明
    2. 使用when标签做判断，相当于case
    3. 使用otherwise标签做其他情况的声明，相当于default

  6. foreach:相当于java代码的for语句

    1. 完成重复操作的属性有：begin：开始值，end：结束值，var：临时变量，step：步长，varStatus:循环状态对象(index:容器中元素的索引，从0开始,count:循环次数，从1开始)

    2. 遍历容器的属性有items:容器对象，var:容器中元素的临时变量，varStatus:循环状态对象(index:容器中元素的索引，从0开始,count:循环次数，从1开始)
